Detailed Description

  • CONFIG MGMT API
    API Function Description
    sf_configMgmt_setParam() Set the configuration parameter.
    sf_configMgmt_getParam() Get the configuration parameter.
    sf_configMgmt_setDefaultConfig() Set the configurations to the default values. The assigned values will be stored persistently into the internal flash.
    sf_configMgmt_writeConfig() Write device configuration into the NVM. This function needs to be called every time sf_configMgmt_setParam is executed to store the new parameter value into the flash.

    sf_configMgmt_getDeviceStatus() Get the device status.

◆ SF_CONFIGMGMT_MEAS_INTERVAL_LIMIT_LOWER

#define SF_CONFIGMGMT_MEAS_INTERVAL_LIMIT_LOWER   (3U)

Measurement interval lower limit. The highest measurement rate i.e. a new measurement at every Tx cycle. As it is impossible to set a timeout of fraction of seconds using CONTIKI event timer, we define the lower limit in seconds.
