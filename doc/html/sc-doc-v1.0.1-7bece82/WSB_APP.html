Wireless Smart Battery application

The WSB application is the recipe of the following features.

  • Join mechanism with button click.
  • Rejoin to the same BMS-CC function in case of connection loss because of SC or BMS-CC restart or bad coverage.
  • Multiple SCs join, up to 30 SCs, to one BMS-CC
  • Transmission of measurement telegrams
  • Persistent storage of device configurations at the SC as well as the BMS-CC and the sensor list storage at the BMS-CC
  • Support of overlapping networks. SCs connect to the network providing the best RSSI level
  • Factory reset function
  • Absolute time synchronization between the SCs and BMS-CC
  • Transmission of downlink messages to the connected SCs
  • Wireless support for glob 2.4 GHz region

Supported devices

  • BMS-CC Battery Management System-Communication Controller:
    • TI LaunchPad CC26x2r1
  • SC Smart Cell:
    • TI LaunchPad CC26x2r1

Hands-On the system

Setup

  1. Flash the BMS-CC hardware as well as the SC hardware with the corresponding firmware.
  2. Reset all target if it is not done automatically by the flashing tool.

Usage

  1. Enable the join window by pressing the button on the BMS-CC hardware. If the join window is active the hardware will turn on the green LED.
  2. The join process is started right away at the SCs after waking up by a button press. Firstly, the green LED will be blinking all along the beacon scan. As the handshake procedure starts, the red LED will be blinking.
  3. If a SC has successfully joined, the green LED shortly flashes, then the measurement mode starts. Measurement telegrams are, then, sent periodically.
  4. The communication status as well as the current slave state are tracked via logs serialized over the virtual serial port of the launchpad.
