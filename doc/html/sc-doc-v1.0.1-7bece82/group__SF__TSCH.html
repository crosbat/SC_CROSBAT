Detailed Description

  • TSCH abstraction API
    API Function Description
    sf_tsch_init() Initialize TSCH.
    sf_tsch_start() Start MAC as end point.
    sf_tsch_stop() Stop MAC.
    sf_tsch_setTxPowerMax() Set Tx power to the maximum value.
    sf_tsch_setDeviceAddress() Set device link address (short address) into the stack.
    sf_tsch_addDataSlots() Add data time-slots for the given sensor address to the schedule. This slots are used for regular communication.
