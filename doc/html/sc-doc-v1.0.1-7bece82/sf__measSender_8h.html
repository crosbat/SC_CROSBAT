void sf_measSender_start (void)
 Kick off the measurement sender module. This triggers tow processes meas_read_process and meas_tx_process. The first process periodically performs new measurement based on the configured measurement interval by calling measHandler_performMeas. The second process controls the transmission time. At every slot cycle it checks for measurement by calling measHandler_getMeas and sends it if available.
